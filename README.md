## Featured Project

### [CINMATCH — Intelligent Movie Recommendation System](https://github.com/chouayb-lkh/CINMATCH)

> A full-stack web application that recommends movies using SVD collaborative filtering and generates personalized explanations using a Large Language Model (Groq / Llama 3.1).

| Feature | Details |
|---------|---------|
| Algorithm | SVD (Singular Value Decomposition) — RMSE: **0.9329** |
| LLM | Groq API — llama-3.1-8b-instant |
| Backend | FastAPI (Python) + JWT Authentication |
| Database | MySQL + SQLAlchemy |
| Frontend | HTML / CSS / JavaScript (8 pages) |
| Dataset | MovieLens 100K + TMDB API (2,519 movies) |
